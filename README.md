# TKF Restaurant Payment System

A console-based **restaurant payment system developed in C** to simulate the ordering and payment process of **The Korean Food (TKF) Restaurant**.

This project represents an enhancement of the original TKF Restaurant Payment System by improving the system functionality and applying better programming practices.

---

## Description

The TKF Restaurant Payment System is a console application that manages a simple restaurant ordering and payment workflow.

The system allows users to:

- View restaurant menu
- Place food orders
- Calculate total payment
- Generate receipts
- Store transaction records

This enhancement focuses on improving the original system through better code organization, improved functionality, and practical application of C programming concepts.
